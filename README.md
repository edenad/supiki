# 🌿 Supiki World

スピキたちが画面の中で自由に動き回るデスクトップマスコット風シミュレーターです。  
ブラウザで動作するので、インストール不要でそのまま遊べます！

## 🎮 遊び方

**▶ [GitHub Pagesでプレイ](https://edenad.github.io/supiki/)**

| ボタン | 効果 |
|--------|------|
| `+ Supiki` | スピキを追加 |
| 🍬 | シナモン（お菓子）を出す |
| 🌵 | サボテンを出す |
| 🔵 | トランポリンを出す |
| 💧 | 池を出す |
| 🎃 | かぼちゃを出す |
| 🧊 | 氷を出す |
| 🥤 | 自動販売機を出す |

- スピキをドラッグして動かせます
- `G` スライダーで重力を調整できます
- `Clear` で全スピキを消去
